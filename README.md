# LEARN BASIC EXPRESSJS x MONGODB

**A simple RESTFULL API using [ExpressJS]('https://expressjs.com'), and [MongoDB]('https://mongodb.com')**

How To Run

	npm i && npm start

List Routes

	GET /api/v1/products
	GET /api/v1/products/:id
    POST /api/v1/product
    PUT /api/v1/product/:id
    DELETE /api/v1/product/:id
